Prediction API

Real-time housing price prediction based on input features.

Automated Retraining

Trigger model retraining on-demand via a simple API endpoint.

GitHub Actions for automated linting, testing, and deployment.

Track experiments with MLflow and monitor system metrics with Prometheus & Grafana.

🚀 Installation Guide

1. Clone Repository

git clone https://github.com/Chandrashekhar-26/california-housing-price-prediction.git
cd california-housing-price-prediction

2. Set Up Virtual Environment (Optional)

# On Ubuntu/macOS
source venv/bin/activate

# On Windows
venv\Scripts\activate

pip install -r requirements.txt

3. Configure Environment

scp .env.prod .env

4. Prepare Dataset

scp .\resources\data\housing_data_raw.csv .\resources\data\housing.csv

▶️ Running the App

Make sure Docker Desktop is running, then use a single command to launch all services.

docker-compose up --build
